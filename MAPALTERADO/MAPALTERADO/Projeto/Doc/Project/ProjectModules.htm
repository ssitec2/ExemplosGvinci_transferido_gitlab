MÓDULOS DO SISTEMA

Os Módulos do sistema serão listados no menu e classificados de acordo com seu tipo.

Página inicial

É a página exibida logo após a tela de apresentação. Essa página é responsável por carregar as telas de configuração de bancos e acesso à aplicação.

Controle de acesso

É utilizado para criar novos usuários e atribuir opções de acesso a determinadas áreas da aplicação, permitindo ou restringindo o uso de suas funcionalidades, de acordo com o usuário.

Login

É exibido quando o usuário tenta acessar áreas restritas. O nome de usuário e senha previamente cadastrados serão necessários para liberar o acesso.

Filtro

Filtra os dados inseridos de acordo com a preferência do usuário, com opções previamente cadastradas oferecidas na operação.

Construtor de Consultas

Oferece ao usuário a possibilidade de personalizar suas próprias consultas (queries) SQL sem que seja necessário um conhecimento profundo dos comandos da linguagem, oferecendo consultas previamente montadas e padronizadas.

Visualizador de consultas

Apresenta as consultas que já foram criadas pelo usuário através do Construtor de Consultas.Permite a reutilização dessas consultas ou a exclusão delas.

Consultas

Apresenta as consultas criadas pelo usuário e permite que o usuário execute a consulta novamente.

Configura DB

Permite a configuração dos bancos de dados que são utilizados pela aplicação, alteração da estrutura da tabela, inclusão de campos e índices, etc.

Informações DB

Apresenta informações sobre os bancos de dados em uso pela aplicação, suas estruturas e integridades, além de outras informações básicas sobre a estrutura do banco.
