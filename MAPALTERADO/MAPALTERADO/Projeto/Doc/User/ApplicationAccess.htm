Acessando a aplicação

O plano de senhas

Quando a aplicação for executada pela primeira vez, é importante fazer o cadastramento dos usuários que irão utilizá-la. O plano de senhas tem a função de receber este cadastramento e classificá-los em grupos.

Senha

Qualquer usuário cadastrado na aplicação tem permissão para alterar sua própria senha, basta seguir as instruções abaixo:

  1. Acesse o módulo de alteração de senha
  2. Clique no campo Senha atual e digite a sua senha, seguida de TAB;
  3. Digite a Nova Senha, no campo apropriado, seguida de TAB;
  4. Redigite a Nova Senha, para confirmar a digitação anterior.

Para confirmar as informações clique sobre o botão aceitar.

Para descartar as informações digitadas, utilize o botão cancelar.

Grupo

O sistema de senhas da aplicação contém um nível para agrupar usuários, que chamamos Grupo, assim, cada usuário deverá estar cadastrado em um grupo. As permissões de operação em cada tabela da aplicação, são definidas para cada um dos Grupos.

Incluindo um novo grupo

  1. Clique sobre o botão de inclusão
  2. Digite o nome do novo grupo no campo Novo grupo
  3. Clique sobre o botão Salvar

Execute a operação abaixo para modificar as permissões de operações aos usuários que serão cadastrados no novo grupo criado.

Modificando permissões de operações nas tabelas

  1. Acesse o menu de Grupos
  2. Selecione o grupo que deseja modificar as permissões
  3. Na moldura intitulada Tabelas e permissões, aparecerá uma janela com os nomes das tabelas existentes
  4. Selecione a tabela desejada
  5. Selecione as opções para marcar ou desmarcar as operações que podem ser permitidas ou proibidas.
  6. Clicar sobre o botão de Salvar

Removendo um grupo

  1. Acesse o menu de Grupos
  2. Selecione o grupo que se deseja remover
  3. Clique sobre o botão de remoção
  4. Será solicitada uma confirmação para a remoção do grupo
  5. Clique sobre o botão de confirmação.

É importante lembrar que:

Toda vez que um grupo é excluído, os usuários que foram cadastrados nesse grupo também são excluídos.

Usuários

Todos os usuários devem ser cadastrados anteriormente ao acesso. O cadastramento de usuário deverá ser feito em um grupo no plano de senhas.

Incluindo um novo usuário

  1. Clique sobre o botão Novo
  2. Escolha o grupo sob o qual o novo usuário deverá ficar vinculado (fique atento às permissões do grupo);
  3. Digite o nome do novo usuário no campo Novo usuário, seguido de TAB
  4. Digite a senha do novo usuário, seguido de TAB;
  5. Redigite a senha, para confirmar, seguido de TAB
  6. Redigite a senha, para confirmar, seguido de TAB
  7. Digite uma observação sobre o usuário
